# CoffeeReview
> 맛집이나 카페를 추천해주는 앱이나 웹페이지는 존재하지만, 해당 카페 내의 메뉴를 추천받으려면 직접 블로그 등 SNS를 통해 찾아야 했기 때문에, 이러한 불편함을 없애고자 웹페이지 개발을 진행했습니다.
